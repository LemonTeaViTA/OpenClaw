## 📋 核心知识点目录

根据文档内容，Redis 面试核心涵盖：

1. **数据结构与使用场景**
   - String、List、Hash、Set、ZSet
   - 底层实现（SDS、跳表、压缩列表）
   - 应用场景（缓存、计数器、排行榜）

2. **持久化机制**
   - RDB 快照
   - AOF 日志
   - 混合持久化

3. **高可用架构**
   - 主从复制
   - 哨兵模式
   - Redis Cluster 集群

4. **性能优化**
   - 缓存穿透/击穿/雪崩
   - 热点 Key 问题
   - 大 Key 拆分

5. **分布式锁**
   - SETNX 实现
   - Redlock 算法
   - 锁超时与续期

## 🎯 学习建议

1. 结合派聪明项目中的缓存场景理解（如 ES 查询结果缓存）
2. 重点关注：缓存一致性策略、分布式锁实现
3. 模拟面试时优先考察：数据结构选型、缓存异常处理、集群原理

## 🔗 与 MySQL 对比

| 特性 | MySQL | Redis |
|------|-------|-------|
| 存储介质 | 磁盘 | 内存 |
| 数据结构 | 表/行 | 丰富类型 |
| 适用场景 | 持久化存储 | 高速缓存 |
| 事务支持 | 完整 ACID | 部分支持 |
