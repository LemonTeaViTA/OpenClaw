## 📝 标准流程（每次复习必做）

### 第 1 步：PDF 内容提取（第 1 次）

```bash
# 提取 PDF 对应章节内容
pdftotext -layout <PDF 文件> /tmp/extract-1.txt
grep -n "题号\|考点" /tmp/extract-1.txt
```

**记录：**
- 提取时间
- PDF 页码范围
- 题目数量

---

### 第 2 步：PDF 内容提取（第 2 次）


**对比检查：**

## 🔧 工具命令

### PDF 提取
```bash
# 提取全文
pdftotext -layout <PDF 文件> /tmp/extract.txt

# 提取特定章节
grep -A 500 "异常处理" /tmp/extract.txt | head -500

# 查找题号
grep -n "^[0-9]*\." /tmp/extract.txt
```

### 对比检查
```bash
# 对比两次提取的题目数量
grep -c "^[0-9]*\." /tmp/extract-1.txt
grep -c "^[0-9]*\." /tmp/extract-2.txt
```
