## 📋 会话元数据

| 字段 | 值 |
|------|------|
| **日期** | YYYY-MM-DD |
| **时间段** | HH:MM-HH:MM |
| **会话 ID** | session-N（当日第 N 次） |
| **主题** | 简短描述（如"JVM Day 1 复习"、"随机复习机制优化"） |
| **标签** | #标签 1 #标签 2 #标签 3 |
