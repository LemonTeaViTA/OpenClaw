## 🎯 用途

每个章节一个独立文件，包含：

- ✅ 完整题目列表
- ✅ 每次答题记录（日期、回答摘要、得分）
- ✅ 复习次数和复习点
- ✅ 薄弱项标记和修复历史
- ✅ 模块汇总统计

---

## 📊 与 MEMORY.md 的关系

| 文件 | 层级 | 内容 | 大小 |
|------|------|------|------|
| `MEMORY.md` | 统计层 | 汇总统计 + 索引 | 几百行 |
| `memory/java-questions/*.md` | 过程层 | 详细答题记录 | 每章 100-200 行 |
| `Obsidian/40-Questions/*.md` | 知识层 | 纯净题库（不含记录） | 每章 200-500 行 |

---

## 📝 使用方式

### 1. 查看某章节的详细答题记录

```
打开 → memory/java-questions/章节名 - 答题记录.md
```

### 2. 查找薄弱项

```
打开 → MEMORY.md → 第 1 节（当前薄弱项）
     → 查看索引 → 跳转到对应章节文件
```

### 3. 更新答题记录

学习新题目后：
1. 在对应章节文件中添加答题记录
2. 更新模块汇总统计
3. 如有薄弱项，同步更新 MEMORY.md 第 1 节
