## 题目 1：Java 中 IO 流分为几种？

**来源：** PDF 题 44 + 美团一面  
**难度：** ⭐⭐⭐  
**考察频率：** 🔥🔥🔥 高频

### 题目
Java 中 IO 流分为几种？/ Java IO 流如何划分？

### 标准答案要点

**四个划分维度：**

1. **按数据流方向**
   - 输入流（Input Stream）：从源读取到程序
   - 输出流（Output Stream）：从程序写出到目的地

2. **按处理数据单位**
   - 字节流：以字节为单位（音频、图像、视频）
   - 字符流：以字符为单位（文本数据）

3. **按功能**
   - 节点流：直接连接数据源（FileInputStream）
   - 处理流：包装已有流（BufferedInputStream）
   - 管道流：线程间传输（PipedInputStream）

4. **按阻塞方式**
   - BIO：阻塞式
   - NIO：非阻塞
   - AIO：异步

**设计模式：** 装饰器模式（处理流包装节点流）

### 面试考察点
- IO 流体系完整理解
- 多维度分类能力

### 题目

### 标准答案要点

**原因：**
1. 字符流由 JVM 将字节转换得到，过程耗时

**使用场景：**
- 字节流：音频、图片、视频等二进制文件
- 字符流：文本文件

## 题目 4：BIO、NIO、AIO 之间的区别？

**来源：** PDF 题 46 + 美团/比亚迪/360/阿里云面试  
**难度：** ⭐⭐⭐⭐⭐  
**考察频率：** 🔥🔥🔥 高频

### 题目
BIO、NIO、AIO 之间的区别？

### 标准答案要点

| 特性 | BIO | NIO | AIO |
|------|-----|-----|-----|
| **全称** | Blocking IO | Non-blocking IO | Asynchronous IO |
| **阻塞方式** | 阻塞 | 非阻塞 | 异步 |
| **引入版本** | JDK 1.0 | JDK 1.4 | JDK 7 |
| **线程模型** | 一个连接一个线程 | 一个线程处理多个连接 | 异步回调 |
| **核心组件** | Stream、Reader/Writer | Channel、Buffer、Selector | AsynchronousChannel |
| **适用场景** | 连接数较少 | 连接数多、时间短 | 连接数多、时间长 |

**BIO：** 阻塞式，线程 IO 操作时被阻塞，基于 Stream/Reader

**NIO：** 非阻塞，通过 Selector 实现多路复用，一个线程处理多个连接

**AIO：** 异步，线程发起 IO 后立即返回，完成时回调通知

### 面试考察点
- 三种 IO 模型深度理解
- 实际应用场景选择

## 题目 7：serialVersionUID 有什么用？

**来源：** PDF 题 47  
**难度：** ⭐⭐⭐⭐  
**考察频率：** 🔥🔥🔥 高频

### 题目
serialVersionUID 有什么用？

### 标准答案要点

**作用：** 标识类版本的唯一标识符，确保序列化和反序列化版本兼容

**规则：**
- 不显式声明：JVM 根据类详细信息自动生成
- 类结构变化时，自动生成的 serialVersionUID 会变化
- 版本不匹配会导致反序列化失败
- **建议：** 显式声明 `private static final long serialVersionUID = 1L;`

### 面试考察点
- 版本控制理解
- 序列化兼容性

### 标准答案

```java

    private String name;
```
