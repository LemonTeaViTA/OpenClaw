## 📌 学习技巧

### 1. 类比记忆法

| JVM 概念 | 生活类比 |
|---------|---------|
| 堆 | 仓库（共享，存货物/对象） |
| 栈 | 工作台（私有，放临时工具） |
| 程序计数器 | 书签（记录读到哪一页） |
| 类加载器 | 图书管理员（把书搬进图书馆） |
| 执行引擎 | 阅读器（读书并理解） |
| GC | 清洁工（清理垃圾） |
| TLAB | 个人工具箱（减少争抢） |

### 2. 流程图记忆

**对象创建流程：**
```
检查类加载 → 分配内存 → 初始化零值 → 设置对象头 → 执行构造
```

**内存区域：**
```
JVM 内存
├── 线程私有（3 个）
│   ├── 程序计数器（无 OOM）
│   ├── 虚拟机栈（StackOverflow/OOM）
│   └── 本地方法栈（StackOverflow/OOM）
└── 线程共享（2 个）
    ├── 堆（GC 主要区域）
    └── 方法区（元空间）
```
