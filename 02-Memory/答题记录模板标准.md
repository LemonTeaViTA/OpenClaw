## 📋 必填字段

**所有答题记录必须包含以下字段**：

### 1. 题目 ID（必需！）
```markdown
**题目 ID**：java-001, java-002, coll-015, jvm-023
```

**ID 格式**：
- `java-XXX`：Java 基础篇（001-056）
- `coll-XXX`：集合框架篇（001-037）
- `jvm-XXX`：JVM 篇（001-066）

**来源**：从题库文件中获取题号

### 2. 题目内容
```markdown
**问题**：...
```

### 3. 得分
```markdown
**得分**：X/5
```

## ⚠️ 禁止行为

- ❌ 答题记录不包含题目 ID
- ❌ 只写题目名称，不写 ID
- ❌ 复习后不同步状态

---

## ✅ 必须行为

- ✅ 每次答题记录必须包含题目 ID
- ✅ 从题库文件中获取正确 ID
- ✅ 复习结束后立即同步状态
- ✅ 检查 todayQuestionIds 是否更新

---

## 🔧 脚本更新

### 同步脚本（sync-review-state-simple.ps1）

**必须能够**：
1. 从答题记录中提取题目 ID（正则：`(java|coll|jvm)-(\d+)`）
2. 更新 `todayQuestionIds` 数组
3. 更新 `excludeIds` 数组
4. 计算平均分

### 随机选题脚本（random-review-selector.ps1）

**必须能够**：
1. 读取 `todayQuestionIds`
2. 从题库中排除已复习的题目
3. 随机抽取 10-15 题
4. 输出题目 ID 和内容
