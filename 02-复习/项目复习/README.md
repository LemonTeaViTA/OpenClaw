## 📋 复习重点

面试中项目相关问题通常围绕：

1. **项目架构设计** — 为什么这样设计？
2. **技术选型理由** — 为什么选这个技术栈？
3. **遇到的挑战** — 遇到过什么问题？如何解决的？
4. **个人贡献** — 你具体负责了什么？
5. **项目亮点** — 有什么值得说的优化/创新？

## 🎯 复习流程

### 1. 项目概述（30 分钟）
- 阅读 `派聪明 rag 项目是什么.md`
- 理解项目定位和核心价值
- 准备 2 分钟自我介绍版本

### 2. 技术架构（1 小时）
- 阅读 `派聪明整体设计方案.md`
- 理解技术选型理由
- 准备"为什么选 Elasticsearch"等问题

### 3. 模块深挖（2-3 小时）
- 选择 2-3 个核心模块
- 阅读对应设计方案
- 准备"遇到的挑战"和"解决方案"

### 4. 面试预测（1 小时）
- 阅读 `架构设计面试题预测.md`
- 自测回答
- 记录薄弱点 → 同步到 `01-薄弱项/weaknesses.md`
