## 🎯 学习流程

### 1. 按分类学习
- 每个分类 10-15 题
- 先学习解题思路
- 再独立实现代码
- 记录时间复杂度和空间复杂度

### 2. 刷题记录
- 记录每道题的解题思路
- 标记薄弱题目（需要二刷、三刷）
- 总结同类题目的解题模板

### 3. 复习策略
- 艾宾浩斯复习点（1 天/3 天/7 天/15 天）
- 薄弱题目优先复习
- 定期模拟面试（手写代码）
