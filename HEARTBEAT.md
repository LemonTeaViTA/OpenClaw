# HEARTBEAT.md - 定期检查任务

## 每日必查 (每次 heartbeat 轮询)
- [ ] **检查早报待发送**：如果 `/root/.openclaw/workspace/.pending-daily-plan.md` 存在，读取并发送给用户，然后删除文件
- [ ] 检查是否有未发送的日报（22:30 前）
- [ ] 检查备份日志 `/root/logs/oc-backup.log` 是否有错误
- [ ] 检查复习进度是否更新（对比 MEMORY.md 和实际进度）

## 每周日额外检查
- [ ] 生成周报复盘（更新 MEMORY.md 第 4 节）
- [ ] 检查下周计划是否需要调整

## 每月末额外检查（28-31 号）
- [ ] **生成月报** → 写入 `memory/YYYY-MM.md`
- [ ] 统计本月学习进度、掌握度变化、薄弱项修复率
- [ ] 制定下月学习计划

## 每年末额外检查（12 月 28-31 号）
- [ ] **生成年报** → 写入 `memory/YYYY-annual.md`
- [ ] 统计全年学习历程、重大突破、成长曲线
- [ ] 制定来年学习规划

## 注意事项
- **早报发送时间：每天 08:00（复习前）** ✅ 已调整
- 日报发送时间：每天 22:30（复习结束后）
- 月报生成时间：每月末 22:30
- 年报生成时间：每年末 22:30
- 备份时间：每天 12:00 和 22:30
- 代码更新：每天 08:00（复习前）
- 如果 heartbeat 发现任务未完成，立即提醒用户

## 重要规则（新增）
- ❌ 禁止在 22:30 之前发送日报
- ❌ 禁止在用户未确认"今天结束"时生成日报
- ✅ 任务完成后必须先询问"是否继续"
- ✅ 备份失败必须立即通知用户
