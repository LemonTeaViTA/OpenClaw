## ⚠️ 故障处理

### 备份失败常见原因
1. **网络问题** - 检查 `ping github.com`
2. **Token 过期** - 更新 `/root/.secure_tokens.sh`
3. **Git 冲突** - 先 `git pull` 再 `git push`
4. **SSH 密钥问题** - 检查 `~/.ssh/config`

### 手动重试
```bash
# 查看失败日志
tail -50 /root/logs/obsidian-backup.log

# 手动执行备份
/root/scripts/sync-obsidian-backup-push.sh

# 如果还是失败，检查网络
ping -c 4 github.com
ssh -T git@github.com
```
