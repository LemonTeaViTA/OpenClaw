# 面试模块

> ⏳ **状态**: 等待简历完成后启用  
> **原因**: 面试官会围绕简历内容进行提问

---

## 🚀 启动条件

- ✅ 简历已完成
- ✅ 项目资料已整理（见 `02-复习/项目复习/`）
- ✅ 八股基础已掌握（薄弱项≤3 个）

---

## 📋 功能规划

### 1. 简历上传/解析
- [ ] 上传简历（PDF/Markdown）
- [ ] 解析简历内容（项目、技术栈、工作经历）
- [ ] 生成面试问题库

### 2. 模拟面试流程
- [ ] 自我介绍环节
- [ ] 项目深挖环节
- [ ] 技术问答环节
- [ ] 反问环节

### 3. 面试复盘
- [ ] 记录回答情况
- [ ] 标记薄弱点
- [ ] 生成改进建议

---

## 🎯 面试问题来源

### 基于简历
- 项目经历相关问题
- 技术栈相关问题
- 工作经历相关问题

### 通用面试题
- 行为面试题（如"你遇到过最大的挑战是什么？"）
- 技术基础题（从八股题库抽取）
- 系统设计题（根据简历技术栈）
